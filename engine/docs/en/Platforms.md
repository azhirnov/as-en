Supported platforms

- [x] Win7 .. Win10
- [x] VisualStudio 2022
- [x] Async file read/write
- [x] Vulkan (by default requires timeline semaphore)
- [x] Vulkan (without timeline semaphore, compile with `-DAE_VK_TIMELINE_SEMAPHORE=OFF`)

- [x] Architecture: x64 
- [ ] Architecture: ARMv8
- [ ] Input: gamepad
